	Exception Handling bisa diartikan menjadi proses yang memungkinkan kita sebagai programmer untuk menulis kode yang dapat menangani kesalahan yang mungkin muncul saat program dijalankan. adanya error yang terjadi di program saat dijalankan dapat menyebabkan program berhenti. untuk membuat program tidak dihentikan maka kita butuh untuk menangkap error menggunakan try dan catch.
	Dalam program yang saya buat menggunakan typeof untuk melakukan validasi tipe data input untuk memeriksa apakah jumlah bayaran cukup. jika dari validasi input terdapat masalah, maka error tersebut akan dilemparkan dengan throw. penggunaan throw error dicontohkan dalam "Error: Both total price and amount given must be numbers" untuk tipe data yang salah dan "Error: Amount given is less than the total price" untuk pembayaran yang kurang. catch digunakan untuk menangkap error yang dilempar dalam program ini, sehingga akan menampilkan pesan yang sesuai dan hal ini mencegah agar program tidak berhenti total. finally juga digunakan agar program selalu dijalankan walaupun terdapat kesalahan yang dicontohkan pada "Cleaning up resources..".
